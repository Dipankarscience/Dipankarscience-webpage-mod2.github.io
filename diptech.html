Concept about Computer Networks

By Dipankar Karmakar

What is Computer Networks?
It is the medium through which communication is established with the help packets by following some protocols. Here packet refers to the data packet consists of header and data part. In header we insert the logic to identify them at receiver side and to detect errors like flag bits, CRC etc.

About layers :
There are different types of layers in computer networks starting from Application layer,Physical layer, Segment layer, Transport layer,Network layer, Data link layer,Physical layer. Data packet comes from the first layer via the medium layers then through physical layer it enters into the network. Different layers have different functions like to segment the packet, to identify the network, to detect whether the particular packet belongs to that particular network or not etc.

Features and elements of computer networks:
There are many more features to control the data and to transport them like there are many protocols as Go-Back-N, selective repeat, sliding window protocol that helps us to transport the data. In collision and collision free mediums we also have many protocols like ALOHA, CSMA,CSMA/CD ,Polling etc. In every situation to handle with we have many protocols. There are routers, switches, bridges as component of computer networks. There is also a cryptography method to provide security while transporting the data.

Need of Computer Networks :
In this modern era we are all in the age of big data. Huge amount of data is there and we have to transport a lot amount of data with a very high security like to encrypt the data and decrypt that same data to read. Cryptography takes place here. And with many more features computer networks provides us all that we need to control that huge amount of data and to transfer them with a very high security and to upload them in a very fast speed that is with a greater amount of band width.
